# 3-tier Architecture

![](https://cdn.jsdelivr.net/gh/AlexLiu2022/resources/img/diagram-of-3-tier-architecture.png)

**三层架构**

- 数据访问层：对数据库的CRUD基本操作
- 业务逻辑层：对业务逻辑进行封装，组合数据访问层层中基本功能，形成复杂的业务逻辑功能
- 表现层：接收请求，封装数据，调用业务逻辑层，响应数据
