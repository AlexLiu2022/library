# Introduction

SpringCloud是目前国内使用最广泛的微服务框架,[官网地址](https://spring.io/projects/spring-cloud)

SpringCloud集成了各种微服务功能组件，并基于Spring Boot实现了这些组件的自动装配，从而提供了良好的开箱即用体验

**SpringCloud 与 SpringBoot 版本间存在依赖关系**
